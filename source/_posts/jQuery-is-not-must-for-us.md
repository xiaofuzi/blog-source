许多刚开始学习js的开发者将jquery当做javascript本身，这其实是一个很大的误解。

在我们学习javascript的时候应该区分开这三者的区别：

* javascript
* Web API
* jQuery(或者是其它的框架)


注意jQuery只是一个库，它并不是javascript本身。这里我们从下面几个方面来对比web API与jquery在使用上的差别。
